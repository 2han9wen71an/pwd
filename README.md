#使用说明

如何正常运行这款项目？ 本项目采用监控执行任务的方式执行密码本用户注册，忘记密码，预约等所 有邮件发送的操作，建议监控时长为1分钟，监控设置为监听器配置

用户管理中心自带的id为1的账号 此账号请勿删除，可以禁用，此账号是作为站长管理员的附属账号，添加的 密码本，备忘录等信息均会附属给id为1的账号，当然如果该账号没有禁用也 是可以单独使用。
